రచయిత పరిచయం
శీలం భద్రయ్య తెలంగాణ ప్రాంతానికి చెందిన కవి, రచయిత. ప్రస్తుతం వృత్తిరిత్యా ప్రభుత్వ తెలుగు ఉపాధ్యాయుడుగా పనిజేస్తున్నారు. ఈయన సూర్యాపేట జిల్లా తుంగతుర్తి మండలం అన్నారం గ్రామంలో 24 డిశంబరు 1980లో జన్మించాడు. శీలం యాదమ్మ, బుచ్చయ్య వీరి తల్లిదండ్రులు. శీలం భద్రయ్య జీవిత భాగస్వామి కరుణ, ఇద్దరు పిల్లలతో ప్రస్తుతం నల్లగొండ పట్టణంలో స్థిరపడ్డారు.
"లొట్టపీసు పూలు, గంగెద్దు" కథాసంపుటాలు రచించాడు.
